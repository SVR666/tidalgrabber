# Tidal Grabber

If you have a lossless subscription, and you want to get .flac files, you can use this python script.

Requirements: 
  - tidalapi
  - transliterate 
  - ffmpeg
  
Script has 3 modes:
  - Playlist grabber
  - Track grabber
  - Album grabber
  - Search
  
Features:
 - Automatically set ID3 tags
 - No need get token manually to use this script
